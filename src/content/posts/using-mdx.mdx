这是一篇 MDX 示例文章（`draft: true`，默认不会在生产构建里出现）。

MDX 让你在 Markdown 中直接写 JSX 表达式，比如算式：{1 + 2 + 3}。

也可以导入并使用 Astro / 框架组件：

{/* import SomeComponent from '@/components/SomeComponent.astro'; */}
{/* <SomeComponent /> */}

平时不需要这些能力时，用 `.md` 就够了。
